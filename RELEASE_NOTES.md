# 딱,가계부 출시노트 📒

## 버전 1.0.0 (2025)

### 🎉 첫 출시

**딱,가계부**는 React Native로 개발된 개인 가계부 관리 앱입니다. Supabase를 활용한 실시간 데이터 동기화와 직관적인 UI/UX로 일상의 지출과 수입을 쉽게 기록하고 관리할 수 있습니다.

---

## ✨ 주요 기능

### 🔐 인증 및 사용자 관리

- **유연한 로그인 방식**

  - 이메일, 닉네임, 아이디 중 하나로 로그인 가능
  - Supabase 기반 안전한 인증 시스템
  - 회원가입 기능 제공

- **마이페이지**
  - 닉네임 수정 기능
  - 로그아웃 기능
  - 회원 탈퇴 기능 (계정 및 모든 데이터 완전 삭제)
  - 사용자 정보 조회

### 📊 대시보드 (메인 화면)

- **이번 달 요약**

  - 총 지출/수입 합계 실시간 표시
  - 일별 지출/수입 차트 시각화 (그라데이션 배경)
  - 사용자 닉네임 표시 ("{닉네임}의 가계부")

- **내역 관리**
  - 이번 달 사용 내역 리스트
  - 하단 플로팅 버튼으로 빠른 추가
  - 리스트 아이템 탭으로 상세 화면 이동

### 💰 내역 등록 및 수정

- **타입 선택**

  - 지출/수입 토글 스위치로 간편 선택

- **상세 정보 입력**

  - 사용 내용(메모) 입력
  - 날짜/시간 선택 (캘린더 UI)
  - 금액 입력
  - 이미지 첨부 (카메라 촬영 또는 갤러리 선택)

- **수정 기능**
  - 기존 내역 수정 가능
  - 이미지 교체 및 삭제

### 📈 통계 및 분석

- **월별 통계 화면**
  - 최근 3개월 또는 올해 전체 요약 선택
  - 월별 지출/수입 막대 그래프 시각화
  - 펼쳐보기/접기 기능으로 공간 효율적 관리
  - 스크롤 가능한 연간 요약 (6개월 단위 표시)

### 🖼️ 미디어 기능

- **이미지 관리**
  - 카메라로 직접 촬영 (`react-native-vision-camera`)
  - 갤러리에서 사진 선택
  - 이미지 미리보기 및 삭제

### 📅 날짜 관리

- **캘린더 선택**
  - 직관적인 캘린더 UI로 날짜 선택
  - 모달 형태의 부드러운 전환 애니메이션

---

## 🎨 UI/UX 개선사항

- **그라데이션 배경**

  - 차트 영역에 세련된 그라데이션 배경 적용
  - 시각적 깊이감 향상

- **반응형 디자인**

  - 다양한 화면 크기 대응
  - 안전 영역(Safe Area) 고려한 레이아웃

- **터치 영역 개선**

  - 아이콘 버튼의 hitSlop 확대로 터치 반응성 향상

- **스플래시 화면**
  - 앱 시작 시 브랜드 이미지 표시
  - 최소 1.5초 표시로 부드러운 시작 경험

## 📱 기술 스택

### 프레임워크

- React 18.2.0
- React Native 0.73.6
- TypeScript 5.0.4

### 주요 라이브러리

- **네비게이션**: `@react-navigation/native`, `@react-navigation/native-stack`
- **데이터베이스 & 인증**: `@supabase/supabase-js`, `@react-native-async-storage/async-storage`
- **UI 컴포넌트**: `react-native-safe-area-context`, `react-native-screens`
- **차트**: `react-native-chart-kit`
- **캘린더**: `react-native-calendars`
- **카메라**: `react-native-vision-camera`
- **갤러리**: `@react-native-camera-roll/camera-roll`
- **아이콘**: `@fortawesome/react-native-fontawesome`
- **SVG**: `react-native-svg`
- **광고**: `react-native-google-mobile-ads`

---

## 📦 컴포넌트 구조

### 재사용 가능한 컴포넌트

- `BannerAdView`: 배너 광고 표시 컴포넌트
- `MonthlySummaryCard`: 월별 요약 카드 컴포넌트
- `MonthlySummaryItem`: 월별 요약 아이템 컴포넌트
- `SplashScreen`: 스플래시 화면 컴포넌트

### 커스텀 훅

- `useAuth`: 인증 관련 로직 (로그인, 로그아웃, 닉네임 수정, 계정 삭제)
- `useAccountBookHistoryItem`: 가계부 내역 관리 로직
- `useInterstitialAd`: 전면 광고 관리 로직
